# DTX Entertainment Explorer

An interactive single-page experience that curates the best venues, restaurants, events, and experiences around Dallas–Fort Worth. The hero pairs an Anime.js-powered heading with a custom Three.js starfield while the directory offers instant filtering by category, neighborhood, rating, and search keywords.

## Features

- **Immersive hero** – Anime.js animates the headline while a custom Three.js scene brings depth to the landing experience.
- **Smart filters** – Toggle categories, set a minimum rating, and filter by neighborhood in real time.
- **Instant search** – Find experiences by name, vibe, or tags like “live music” or “family-friendly”.
- **Trending tags** – Quick shortcuts into the most popular themes for the metroplex.
- **Curated highlights** – Suggested itineraries to help you plan an unforgettable trip.

## Getting started

No build step is required. Simply open `index.html` in a modern browser or serve the directory with any static file server.

```bash
# Example using Python
python -m http.server 3000
```

Then visit [http://localhost:3000](http://localhost:3000) to explore the experience directory with live filtering and animations.
