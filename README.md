# DTX Entertainment Explorer

A single-page web application that curates the best venues, restaurants, events, and experiences around Dallas–Fort Worth. Travelers can quickly search and filter by neighborhood, category, and rating to build the perfect itinerary.

## Features

- **Smart filters** – Toggle categories, set a minimum rating, and filter by neighborhood.
- **Instant search** – Find experiences by name, vibe, or tags like “live music” or “family-friendly”.
- **Trending tags** – Quick shortcuts into the most popular themes for the metroplex.
- **Curated highlights** – Suggested itineraries to help you plan an unforgettable trip.

## Getting started

No build tools are required. Open `index.html` in any modern browser to explore the experience directory.
